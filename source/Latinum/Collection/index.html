def <<(object)

Add a resource, an array of resources, or another collection into this one.

Signature
parameter　object　Resource | Array(Resource) | Collection

The resource(s) to add.

Implementation

 
def <<(object)
	case object
	when Resource
		add(object)
	when Array
		object.each { |resource| add(resource) }
	when Collection
		object.resources.each { |name, amount| @resources[name] += amount }
	end
	
	return self
end

Signature

Implementation

 
def compact
	collection = self.class.new
	
	@resources.each do |key, value|
		unless value.zero?
			collection.resources[key] = value
		end
	end
	
	return collection
end
